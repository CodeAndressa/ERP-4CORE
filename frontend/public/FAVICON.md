# Favicon 4Core

Coloque os arquivos de favicon nesta pasta:

- `favicon.ico` (obrigatório - 32x32px)
- `favicon-32x32.png`  
- `favicon-16x16.png`  
- `apple-touch-icon.png` (180x180px)

O Vite serve esta pasta na raiz do site automaticamente.
Gerador recomendado: https://realfavicongenerator.net/
