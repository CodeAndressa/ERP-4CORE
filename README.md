# 4Core Consultoria Estratégica

Plataforma interna leve para gestão financeira, comercial, clientes, marketing, conhecimento institucional e recomendações com IA.

## Visão geral
Esta solução utiliza Python + FastAPI no backend e React + TypeScript no frontend, com SQLite no MVP e arquitetura preparada para PostgreSQL no futuro.

## Módulos iniciais
- Autenticação e proteção de rotas
- Dashboard executivo
- Financeiro básico
- CRM de leads
- Clientes e propostas
- Marketing editorial
- Base de conhecimento
- IA com recomendações mockadas
- Relatórios em PDF

## Estrutura proposta
- backend/: API FastAPI, SQLAlchemy, Alembic, migrations e seed
- frontend/: app React/Vite/Tailwind/Shadcn UI
- docs/: arquitetura, modelo de dados, rotas e plano de implementação
